
# contedit (PxPlugin)


## インストール方法

1. Pickles Framework をセットアップする。
2. ディレクトリ plugins/contedit を、Pickles Framework の plugins ディレクトリにアップロードする。


## 使い方

1. ブラウザで、PxCommand "?PX=plugins.contedit" にアクセスする。
2. エクセルファイルをアップロード(またはダウンロード)する。
